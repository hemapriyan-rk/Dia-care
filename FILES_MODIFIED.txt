KEY ENDPOINTS CREATED:
======================
POST /daily-log
  - Saves daily data
  - Invokes all 3 Python models
  - Returns prediction immediately

GET /output/latest
  - Get today's prediction

GET /output/history
  - Get prediction history

PYTHON MODELS USED:
===================
1. local.py - Adaptive baseline tracking
   - Input: daily data
   - Output: daily_deviation, cumulative_deviation, signal_ready

2. global_infer.py - LightGBM population model
   - Input: user demographics + daily data
   - Output: population_glucose_deviation_z, population_deviation

3. final.py - Combined prediction
   - Input: local output, global output, account age
   - Output: final_deviation_score, final_deviation, weights

INTEGRATION FLOW:
=================
User → Frontend (DailyEntryPage)
     ↓
POST /daily-log with daily data
     ↓
Backend saves to daily_behavior_logs
     ↓
Fetches user profile & baseline
     ↓
Invokes local.py (adaptive baseline)
     ↓
Invokes global_infer.py (LightGBM)
     ↓
Invokes final.py (combine models)
     ↓
Stores all outputs in 3 tables
     ↓
Returns prediction to Frontend
     ↓
DailyEntryPage shows result → navigate to Dashboard
     ↓
Dashboard displays:
  - Today's status card
  - Risk zone + deviation score
  - Explanation text
  - 7-day prediction history chart
